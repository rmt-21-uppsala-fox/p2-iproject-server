# Job Portal API Documentation

## Endpoints :

List of available endpoints:

- `POST /register`
- `POST /login`
- `POST /recipes`
- `GET /recipes/filter`
- `POST /jobs`
- `GET /jobs`
- `GET /jobs/:id`
- `PUT /jobs/:id`
- `DELETE /jobs/:id`
- `PATCH /jobs/:id`
- `GET /companies`
- `GET /histories`
- `GET /users/bookmarks/:userId`
- `POST /users/bookmarks/:jobId`
